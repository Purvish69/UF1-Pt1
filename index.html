Font-family

La propiedad `font-family` en CSS se utiliza para especificar la fuente o grupo de fuentes que se aplicarán al texto de un elemento.

Color

La propiedad `color` se utiliza para definir el color del texto dentro de un elemento.

Text-align

La propiedad `text-align` en CSS se utiliza para alinear el texto horizontalmente dentro de un elemento en bloque.

Font-size

La propiedad `font-size` en CSS se utiliza para establecer el tamaño de la fuente del texto en un elemento.

Letter-spacing

La propiedad `letter-spacing` en CSS se utiliza para ajustar el espacio entre los caracteres de un texto.
